Cansado de ser um nerd incompleto? Seus problemas acabaram! Com o Search Hero você vai descobrir TUDO sobre seus personagens favoritos da Marvel Comics™ / Tired of being a incomplete nerd? Your problems are OVER! With Search Hero you'll discover EVERYTHING about your favorites characters of Marvel Comic's™️

Search Hero is a frontend web application designed to serve as a comprehensive encyclopedia for Marvel Comics characters. It allows users to search, discover, and view detailed information about their favorite heroes and villains by seamlessly integrating with the official Marvel API.

![](/screen.png)

## ⚙️ Technologies Used / Tecnologias Utilizadas

This project leverages a modern React ecosystem tailored for high performance, maintainability, and excellent user experience:

* **[React 18](https://reactjs.org/)** - Core UI library
* **[Vite](https://vitejs.dev/)** - Extremely fast frontend tooling and bundler
* **[TypeScript](https://www.typescriptlang.org/)** - Strict static typing for JavaScript
* **[Styled Components](https://styled-components.com/)** - CSS-in-JS for component-scoped styling
* **[React Router DOM](https://reactrouter.com/)** - Declarative routing for Single Page Applications
* **[Axios](https://axios-http.com/)** - Promise-based HTTP client
* **[Swiper](https://swiperjs.com/)** - Modern mobile touch slider
* **[Jest](https://jestjs.io/) & [React Testing Library](https://testing-library.com/)** - Robust unit and component testing framework
* **[MirageJS](https://miragejs.com/)** - API mocking for development and testing environments
* **[ESLint](https://eslint.org/) & [Prettier](https://prettier.io/)** - Code linting and formatting

## 📋 Prerequisites / Pré-requisitos

Before running this project, ensure you have the following installed:
* [Node.js](https://nodejs.org/) (Latest LTS recommended)
* npm (comes with Node.js)

You will also need to configure your environment variables to connect to the Marvel API:
1. Copy the `.env.example` file and rename it to `.env`.
2. Add your Marvel API Public and Private Keys to the newly created `.env` file.

## 🚀 How to Run / Como Executar

```bash
# Install dependencies / Instalar dependências
npm install

# Run in development mode / Rodar em desenvolvimento
npm run dev

# Build for production / Fazer build de produção
npm run build

# Preview production build / Visualizar build de produção
npm run preview

# Run tests / Rodar testes
npm test

# Run linter / Rodar linter
npm run lint
```
